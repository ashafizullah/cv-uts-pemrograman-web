# CV Website — Adam Suchi Hafizullah

Website **Curriculum Vitae (CV)** pribadi berbasis web statis yang menampilkan ringkasan profil, keahlian teknis, pengalaman profesional, dan pendidikan. Dibangun dengan **HTML5, CSS3, JavaScript, dan Bootstrap 5** dengan desain bersih, minimalis, dan responsif.

Proyek ini dikerjakan untuk **Ujian Tengah Semester (UTS) — Pemrograman Web I**.

🔗 **Live demo:** [https://cv.ashafizullah.com](https://cv.ashafizullah.com/)

## Teknologi

- **HTML5** — struktur konten semantik
- **CSS3** — styling kustom (tema bersih & minimalis, fully responsive)
- **JavaScript** — interaktivitas ringan (navigasi & efek scroll)
- **Bootstrap 5** — layout grid & komponen
- **Bootstrap Icons** & **Google Fonts** (Inter, Lora)

## Struktur Proyek

```
project-cv/
├── index.html              # Halaman utama
├── README.md
└── assets/
    ├── css/style.css       # Styling kustom
    ├── js/script.js        # Interaktivitas
    └── img/
        ├── profile.jpg     # Foto profil
        ├── logo.svg        # Logo monogram
        └── favicon.svg     # Ikon situs
```

## Fitur

- Desain responsif (mobile, tablet, desktop)
- Navigasi satu halaman (single-page) dengan smooth scroll
- Bagian: Beranda, Tentang, Keahlian Teknis, Pengalaman, Pendidikan, dan Kontak
- Tabel keahlian & riwayat pengalaman kerja
- Tautan ke profil LinkedIn, GitHub, dan website pribadi

## Menjalankan Secara Lokal

Buka `index.html` langsung melalui browser, atau jalankan server statis:

```bash
python3 -m http.server 8080
# akses di http://localhost:8080
```
